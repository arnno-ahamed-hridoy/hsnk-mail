## ✨ Overview

**HSNK Mail** is a lightweight, browser-based disposable email service built for anyone who needs a quick, private inbox — for sign-ups, verifications, testing, or avoiding spam. No installation, no account creation on your end, and no personal data collected.

Just open the site, pick a name, and your inbox is ready in seconds.

## 🚀 Features

- **Custom or random addresses** — type any name you like, or generate one instantly
- **Multiple domain options** — choose from a growing list of available domains
- **Reusable inbox** — reopen the same address later on the same browser and pick up right where you left off, no login screen
- **Live auto-refresh** — new emails appear automatically, no manual reload needed
- **One-click copy** — grab your address instantly to paste anywhere
- **Clean, distraction-free interface** — built for speed on both mobile and desktop
- **No signup, no personal data stored on any server**

## 🖥️ How It Works

1. Open the site
2. Type a name (or hit **New random name**)
3. Pick a domain from the dropdown
4. Tap **Get this address** — your inbox is live
5. Use that address anywhere you need a temporary email
6. Incoming messages appear automatically — tap any message to read it in full

Your address and its access credentials are stored securely in your own browser, so returning to the same address later reconnects you automatically — nobody else can access your inbox.

## 🛠️ Tech Stack

- **Pure HTML, CSS & JavaScript** — no frameworks, no build step, no dependencies
- **[mail.tm](https://mail.tm)** & **[mail.gw](https://mail.gw)** — free public temporary-email APIs powering message delivery
- Hosted for free on **GitHub Pages**

## 📦 Getting Started (Self-Hosting)

Want to run your own copy?

```bash
git clone https://github.com/arnno-ahamed-hridoy/hsnk-mail.git
```

Then simply open `index.html` in your browser — that's it. No build tools, no server setup required. To publish it, upload the file to any static host (GitHub Pages, Netlify, Vercel, etc.).
